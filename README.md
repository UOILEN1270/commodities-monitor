# Commodities Price Monitor

全球大宗商品价格监控面板，自动更新。

**覆盖品种**: 贵金属(黄金/白银/铂金/钯金) · 工业金属(铜/铝/锌/镍/铅/锡) · 黑色系(螺纹钢/热卷/铁矿石) · 新能源(碳酸锂) · 能源(WTI/布伦特/天然气)

**数据来源**: COMEX/NYMEX (via Yahoo Finance) · 东方财富 (国内期货) · Frankfurter API (汇率)

**更新频率**: 每日 3 次 (亚盘收盘 / 国内收盘 / 美盘收盘)，GitHub Actions 自动执行。

**访问**: https://uoilen1270.github.io/commodities-monitor
